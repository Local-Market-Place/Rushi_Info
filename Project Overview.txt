Local Market Place is a free platform designed to allow local businesses and service providers to easily interact with customers and exchange product and service details. The Web Application provides  product information and comparisons within a 2-km radius shop. It allows you to place and deliver orders, view service provider information with scheduling options, get new shop updates, and use a product inquiry system. It also provides a rapid order and collection service, allowing customers to examine store information, make purchases online, and pick them up without waiting. The platform allows businesses to post and promote for free, and it offers features like an inventory management system.Additionally, an online payment gateway is available for secure transactions.
Front-End: HTML5, CSS3, jQuery, ReactJs, Axios.
Back-End: ASP.NET Web API
Database: SQL Server
Platform: AWS Cloud
